地址选择器组件

可以选择国内的省、市、区三级，返回地区名称（在input框中直接显示，js通过读取value获得），返回地区码（定义在data-code自定义属性中，js读取属性获得）

Demo

Code-html

Code-js

					var address = document.getElementById('address');
					//获取地址名称
					var addr-name = address.value;
					//获取地址编码
					var addr-code = address.getAttribute('data-code');
